# MagicWindows v0.4.0

The biggest release yet: a brand-new download website, a post-install Health
Check, F12/Eject remapping with a Bluetooth pairing assistant, and a key fix
that unblocks Shift+Enter in Chromium-based apps.

*La plus grosse mise à jour à ce jour : un nouveau site de téléchargement, une
page de diagnostic post-installation, le remappage F12/Eject avec assistant
d'appairage Bluetooth, et un correctif important qui débloque Shift+Entrée dans
les applications Chromium.*

## What's new | Nouveautés

### Shift+Enter fixed in Chromium-based apps

Layouts now emit a carriage return (CR) for `Shift+Enter` instead of a line
feed (LF). The old behaviour broke newline insertion in Chromium-based apps
(VSCode terminals, Electron chat inputs, Discord, etc.). Reinstall your layout
to pick up the fix.

*Les layouts émettent désormais un retour chariot (CR) pour `Shift+Entrée` au
lieu d'un saut de ligne (LF). L'ancien comportement cassait l'insertion de
nouvelles lignes dans les applications Chromium (terminaux VSCode, champs de
chat Electron, Discord…). Réinstallez votre layout pour bénéficier du correctif.*

### Health Check — verify your install in one click

A new diagnostic page checks that every key on the installed layout produces
the expected character, with per-key visual feedback on the keyboard, plus a
one-click control-key assertion (Enter, Shift+Enter, Tab, Backspace, Esc) that
catches Chromium-breaking regressions. Results are exportable as JSON.

*Une nouvelle page de diagnostic vérifie que chaque touche du layout installé
produit le bon caractère, avec un retour visuel touche par touche, plus une
assertion en un clic sur les touches de contrôle (Entrée, Shift+Entrée, Tab,
Retour arrière, Échap). Résultats exportables en JSON.*

### F12 / Eject remap + Bluetooth pairing assistant

Pick what the F12/Eject key does from a curated dropdown, and use the new
Bluetooth pairing assistant — Apple keyboards are auto-detected and enumerated
so you can pair without leaving the app.

*Choisissez l'action de la touche F12/Eject dans une liste déroulante, et
utilisez le nouvel assistant d'appairage Bluetooth — les claviers Apple sont
détectés et énumérés automatiquement pour un appairage sans quitter l'app.*

### A real download website

MagicWindows now has a web home at **https://magicwindows.mindvisionstudio.com**
— EN/FR, responsive, with a live keyboard preview. Each layout can be
downloaded as a ready-to-run ZIP (x64 + ARM64) straight from the site, served
from `dl.mindvisionstudio.com` with SHA-256 integrity in the manifest.

*MagicWindows a désormais un site web (**https://magicwindows.mindvisionstudio.com**),
EN/FR et responsive, avec aperçu du clavier en direct. Chaque layout se
télécharge en ZIP prêt à l'emploi (x64 + ARM64) directement depuis le site,
servi par `dl.mindvisionstudio.com` avec intégrité SHA-256 dans le manifest.*
